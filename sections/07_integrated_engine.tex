\section{Integrated MD Engine}

\subsection{Overview}

The six modules developed independently are assembled into a single, functional MD engine. The data flow is:

\begin{enumerate}
    \item \textbf{Module 2} generates the FCC lattice and box length $L^*$.
    \item \textbf{Module 5} initialises velocities from the Maxwell-Boltzmann distribution and zeros net momentum.
    \item \textbf{Module 4} builds the initial Verlet neighbor list.
    \item The main loop iterates:
    \begin{itemize}
        \item \textbf{Module 3} (Velocity Verlet) advances positions and velocities.
        \item \textbf{Module 4} rebuilds the neighbor list when the displacement threshold is exceeded.
        \item \textbf{Module 5} applies velocity rescaling during equilibration.
        \item \textbf{Module 1} provides the force and energy for every interacting pair.
    \end{itemize}
    \item In the production phase, \textbf{Module 6} collects $g(r)$ snapshots every 10 steps.
\end{enumerate}
